Базовое оформление текста на странице

Тэги <h1> - <h6> - отвечают за оформление и вывод заголовков с 1 по 6 уровни.

Тэг <р> - оформляет текст в абзацэ.

Тэги <b> <strong> - эти тэги делают текст жирным.

Тэги <i> <em> - эти тэги делают текст курсивом.

Тэг <s> - этот тэг делает текст зачерунутым

Тэг <u> - этот тэг делает текст подчеркнуты.

Тэги <h1>, <h2>, <h3>, <strong>, <em> - они важны для поисковых систем. Используются для оптимизации страниц сайта под поисковые запросы. Применяйте их с умом
